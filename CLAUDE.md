# Proyecto

Proyecto de configuración para el editor de Neovim con el lenguaje Lua.

## Estructura

- `init.lua`: Archivo de inicio del editor.
- `lua/`: directorio con las diferentes configuraciones y plugins del edito.
- `lua/config/`: directorio con las configuraciones del editor.
- `lua/plugins/`: directorio con la configuración de los plugins.
- `lua/config/autocmd.lua`: archivo con los cmds personalizados.
- `lua/config/options.lua`: archivo con las opciones de configuración del editor personalizadas.
- `lua/config/keymaps.lua`: archivo con las combinaciones de teclas para ejecutar comandos personlizadas.
